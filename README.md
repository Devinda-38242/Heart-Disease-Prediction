# Heart Disease Prediction Project

## Objective
This project predicts whether a patient has heart disease using machine learning.

## Dataset
Heart Disease dataset containing medical attributes such as age, cholesterol, blood pressure, etc.

## Model Used
Random Forest Classifier

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
